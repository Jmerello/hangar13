# Hangar Input

Enterprise input field used throughout Hangar 13.

## Features

- Labels
- Required fields
- Success state
- Warning state
- Error state
- Disabled state
- Helper text
- Built-in validation helpers

## Example

```html
<div class="h13-input-group">

<label class="h13-label">

Customer Name

</label>

<input
class="h13-input"
type="text">

</div>
```
