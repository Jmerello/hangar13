# Hangar Switch

Enterprise toggle switch.

---

## Features

- Modern sliding animation
- Keyboard accessible
- Focus ring
- Disabled state
- JavaScript API

---

## JavaScript

```javascript
const sw=document.getElementById("bayAvailable");

H13Switch.toggle(sw);
```
